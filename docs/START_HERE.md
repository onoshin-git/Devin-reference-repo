# 開始ガイド

このリポジトリは **参考用の GitHub リポジトリ** です。  
実際の構築に使う場合は、この repo の内容を参考にしながら、正本となる contract を別途作成してください。

## このリポジトリで分かること

- Devin に渡すべき情報の分類
- 画面契約、API 契約、データ契約の考え方
- deploy 契約や repair 契約をどう定義するか
- reference と authoritative をどのように分離するか

## このリポジトリで分からないこと

- 本番構築用の exact schema
- 本番構築用の HTML / JS / handler 実装
- 本番環境用の AWS 設定値

## 注意

`references/` は **参考用 stub** です。  
そのまま正本として使わず、必要に応じて authoritative な文書や repo に移してください。
